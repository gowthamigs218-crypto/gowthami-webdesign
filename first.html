Welcome To Smart Shoe Project

An IOT- based wearable safety device that provides real-time GPS tracking,
heart rate monitoring, fall detection, and emergency SMS alerts.

GPS Tracking

Provides real-time location tracking using GPS to help locate the user during emergencies.

Health Monitoring

Continuously monitors heart rate using a pulse sensor to dectect abnormal conditions.

Emergency Alert

Detects falls and sends an emergency SMS with live location to family members.
